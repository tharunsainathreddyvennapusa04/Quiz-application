# Gqt-Project1

A simple Java console version of **Kaun Banega Crorepati (KBC)**.

This project asks multiple-choice general knowledge questions and tracks prize money as the player progresses through levels.

## Features
- 10 multiple-choice questions
- Prize ladder from Rs. 1,000 to Rs. 3,20,000
- Two lifelines:
  - 50-50
  - Audience Poll
- Single-player console gameplay with instant result feedback

## Tech Stack
- Java (Core Java)
- Console-based UI

## Project Structure
- `KBC-Game/SimpleKBC.java`: main game source code
- `ProjectReport.pptx`: project presentation/report

## How to Run
1. Open terminal in project root.
2. Compile:
   `javac KBC-Game\\SimpleKBC.java`
3. Run:
   `java -cp KBC-Game SimpleKBC`
